# JSON Spirit

(Work in progress)

## Build requirements

- Compiler with almost full C++14 support
    - GCC 7.0 or higher
    - Clang 4.0 or higher
    - MSVC 2015 or higher
- Boost 1.59 or higher (Boost.Spirit and Boost.Locale)
- CMake 3.8 or higher

>NOTE: original [JSON Spirit from CodeProject](https://www.codeproject.com/Articles/20027/) has lower requirements
